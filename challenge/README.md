Fix My Code Challenge


This project is part of the Holberton School curriculum.
The goal is to fix existing code instead of rewriting it from scratch.

Each task contains buggy code that needs to be analyzed and corrected.

0. FizzBuzz

Fix a Python implementation of FizzBuzz:

Multiples of 3 → Fizz
Multiples of 5 → Buzz
Multiples of both → FizzBuzz

✔ Fixed by correcting the order of conditions.

1. Print Square

Fix a JavaScript program that prints a square.

2. Sort

Fix a Ruby script that sorts arguments.

3. User Password

Fix a Python script that validates user passwords.

4. Double Linked List

Fix a C program that works with a double linked list.
